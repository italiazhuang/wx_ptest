Running the WebSurvey Sample:

1.  Unzip the WebSurvey.zip archive (this you've already done I see)
2.  Copy the WebSurvey directory into your Web Site root
    For example:
    If your Web Site root is C:\Inetpub\wwwroot\ then copy WebSurvey folder C:\Inetpub\wwwroot\WebSurvey

3.  Open up your web browser, and browse to http://localhost/WebSurvey/
